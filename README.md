# 📊 HR Analytics & Payroll Dashboard - 2026
An interactive Power BI dashboard designed to analyze workforce demographics, absenteeism, and payroll/financial audits for a construction company.

## 🖥️ Live Preview & UI/UX Design
> **Note:** The layout features a customized minimalist top navigation bar to mimic a modern web-application style, maximizing screen space and user experience.

![Dashboard Preview](Screenshots/your_main_image.png)

## 🎯 Key Features & Insights
* **Effectif & Démographie:** Comprehensive overview of total workforce (1K), average seniority (4 years), and contract distribution (CDI, CDD, ANAPEC).
* **Absentéisme & Heures Supp:** Tracking operational strain by monitoring total absence days (350 J) and overtime hours (10,552 H) across different sites.
* **Audit Financier & Masse Salariale:** Financial audit visualizer comparing Net Salaries vs Total Company Cost per site to optimize budget allocation.

## 🛠️ Tools & Technologies Used
* **Power BI Desktop:** Dashboard design, custom layouts, and interactive visual structures.
* **Power Query:** Data cleaning, merging PDF reports, and data transformation.
* **DAX (Data Analysis Expressions):** Created advanced measures for dynamic KPIs and financial aggregations.
